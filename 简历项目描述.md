项目名称：餐饮多平台经营数据分析系统｜个人项目
项目展示：https://restaurant-analytics-8fgygzvdk9b9edpxe4rpdc.streamlit.app/

项目简介：针对餐饮商家多平台订单数据难以有效分析的问题，开发Web端数据分析平台，实现数据上传、自动清洗、多维分析、营收预测及经营建议生成的完整闭环。

核心职责：
• 基于 Python + Streamlit + Pandas + Scikit-learn 搭建数据分析平台，独立完成数据处理、算法建模及公网部署；
• 设计 ETL 数据管道，实现多平台 CSV/Excel 自动识别、缺失值处理及 IQR 异常值检测；
• 构建 RFM 用户分层模型（三维打分→8类分层）与 Apriori 关联规则模型（Support/Confidence/Lift），挖掘高价值用户与菜品搭配规律；
• 将时间序列转化为监督学习问题，构造17维时间特征，基于随机森林回归预测未来营收，MAPE < 9%；
• 集成 Isolation Forest 多维异常检测与 Plotly 可视化，一键生成自然语言经营诊断报告。

项目成果：覆盖数据挖掘、机器学习、异常检测三类共6种算法模型；支持美团、饿了么、微信点单等多平台数据统一分析；已部署至 Streamlit Cloud 公网可访问。
